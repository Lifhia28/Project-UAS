# api-service-sederhana-php
Proyek UAS ini merupakan implementasi REST API sederhana yang dirancang untuk mengelola data pengguna. API ini menyediakan fitur-fitur dasar seperti menampilkan seluruh data user, menambahkan user baru, memperbarui data user berdasarkan ID, hingga menghapus data user.
